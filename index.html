⚠ Limitations

Magic number detection is useful but not a full malware scan. A file can be polyglot, malformed, encrypted, truncated, or intentionally crafted to evade simple checks.

  • The browser-reported MIME type and the file extension are treated as claims, never as proof.
  • Text-based formats (TXT, CSV, JSON, XML, HTML, Markdown) have no strong magic bytes and are detected heuristically.
  • Unknown and ambiguous results are valid outcomes and are shown honestly rather than guessed.

● About

NeonSig inspects a file's raw bytes — its magic numbers and container structure — to determine the actual file type, then compares that against the claimed extension to surface mismatches. Everything runs locally in your browser: no server, no tracking, no CDN, no dependencies. It's a single self-contained index.html.
